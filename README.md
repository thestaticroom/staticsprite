# StaticSprite

A DIY, pocket-sized digital compaion project. Build your own sprite, keep it alive, and customize its hardware.

## Project Tiers

The StaticSprite is designed in two tiers to accommodate different skill levels and tool access.

| Tier | Build Style | Features | Skill Level |
| :--- | :--- | :--- | :--- |
| **Tier 1** | Solderless | Core functionality, basic sprite interaction | Beginner / No tools required |
| **Tier 2** | Soldered | Core functionality + LED lighting arrays | Intermediate / Soldering iron required |

## Community Guidelines & Lore

This project is open source under the GPLv3 license, but we have a few social contracts for the community:

1. **Credit:** If you build, modify, or share your own StaticSprite, please mention **The Static Room** in your resources or credits.
2. **Naming:** Please **do not** name your custom sprites "Signal". That name is reserved for the canonical lore. 

## Files & Assets

- `bom/` - Bill of Materials for Tier 1 and Tier 2 builds.
- `assets/` - Art and visual references, including the Signal sprite.
- `cad/` - 3D printable case files (Coming Soon).
